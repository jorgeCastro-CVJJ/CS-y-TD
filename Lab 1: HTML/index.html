    ¿Cuál es la diferencia entre interner y la World Wide Web?
      Internet es una red de computadoras interconectadas alrededor de todo el mundo. Por otra parte, la World Wide Web es una colección de páginas que existen dentro de esa red de computadoras.

    ¿Cúales son las partes de un URL?
      El protocolo (como http), subdominio, dominio, (TLD) Top Level Domain, subcarpeta/directorio , Parámetro, Etiqueta.

    ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    1. GET: solicita un recurso del servidor web
    2. HEAD: solicita solo el encabezado, sin el archivo
    3. POST: sirve para enviar al servidor web paquetes grandes de datos, adjuntandose en el encabezado html
    4. PUT: modifica todo el recurso completo
    5. PATCH: modifica parte de un recurso
    6. DELETE: eliminar un recurso

    ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
      HTML5 separa los componentes: estilo, contenido y comportamiento

    ¿Qué componentes de estructura y estilo tiene una tabla?
      Tanto en HTML4 como HTML5 existe el tag table, tr (table row), td (table data). Sin embargo en HTML5 se incorporan etiquetas semánticas como th (table header) y tfoot (el footer de la tabla). Esto es una ventaja por que el orden de los tags en el código html no importa, dado a que son etiquetas semánticas.

    ¿Cuáles son los principales controles de una forma HTML5?
      Se creo un elemento input con más capacidades para poder recolectar de mejor manera los datos, manejar errores y validación de errores de mejor manera.

      Ejemplos: type="number", type="range", type="date", type="color", type="search". Se creo de igual manera el tag datalist, para que al llenar un formulario se pueda obtener una lista de donde el usuario pueda seleccionar una.

      Se creo también un elemento placeholder para tener un contenido predeterminado en el campo de entrada de datos.
